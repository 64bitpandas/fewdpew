About TowerRush

Interested in the game? Sign up for future updates!
  • Sign Up!


    • TowerRush is currently being developed with zero profits. If you like our project, please support it by clicking the donate button below. Thank you!

    Unique models

    Every model in the game is created using Blender, an open-source 3D sculpting tool. No model packs or addons used.

    Multiplayer support

    This game uses Photon Unity Network (PUN), so you can play with your friends any time!

    Open source

    During the game's alpha stage, the source code for TowerRush is available for you to download and edit however you want! Take a look at the GNU General Public License for more information.

    Progress Update & Name Change

    6/24/16

    We have decided to change the game's name to TowerRush. Updated logo and splash text coming soon! What else is coming soon? 1.2!!! We are removing much of the placeholder items and scripts found in previous versions and adding some cool models and animations. Kits will also be implemented at least partially next update. We're still trying to get all the basics into the game as quickly as possible- once the framework is complete we can start to polish the game to its final state. But until then, stay tuned for more updates!
