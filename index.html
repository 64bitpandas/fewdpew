About Bumrush

Bumrush is currently being developed with zero profits. If you like our project, please support it by clicking the donate button below. Thank you!

Unique models

Every model in the game is painstakingly shaped using Blender, an open-source 3D sculpting tool. No prefabs, model packs, or addons used.

Multiplayer support

This game uses Unity Multiplayer, so you can play with your friends any time!

Open source

Until the game's full release, the source code for Bumrush is available for you to download, edit, and share however you want! Take a look at the GNU General Public License for more information.
